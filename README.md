# Finance Behavior Tracker

A CLI-based Python application that helps users track expenses and analyze spending behavior.

## Features

- Add expenses
- View expenses
- Total spending analytics
- Category-wise summaries

## Tech Stack

- Python
- Git
- GitHub

## Product Vision

Most expense trackers only store transactions.
This project focuses on helping users understand spending behavior and spending patterns.

## Future Improvements

- JSON data persistence
- Budget alerts
- Weekly reports
- Spending insights
- Data visualization
